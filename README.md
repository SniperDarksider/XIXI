#兮兮（Xixi） -- a cute girl using Python with AIML

## 概述
兮兮是一个用Python实现的小伙伴。
她能撒娇卖萌，她能成为你忠实的小秘书，她能陪你聊天为你唱歌给你读书读新闻。
如果她不乖的话，你还可以调教她。
